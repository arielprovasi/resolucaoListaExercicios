# 1. O que é blockchain?

Blockchain é como um registro digital compartilhado, onde várias pessoas podem ver as informações, mas ninguém consegue alterar depois que foi registrado. Ele é descentralizado, ou seja, não depende de uma empresa ou servidor central.

# 2. Exemplos na Web 2.0 e Web 3.0

Na Web 2.0, que é a internet que a gente usa hoje, tudo é centralizado.
Exemplos: Instagram e YouTube, onde os dados ficam nas empresas e a gente só usa.

Já na Web 3.0, entra o blockchain.
Por exemplo, o Bitcoin usa blockchain para fazer transações sem precisar de banco, e o Ethereum permite criar contratos inteligentes que funcionam sozinhos.

# 3. A Web 3.0 já está pronta?

Ainda não. Ela ainda está em desenvolvimento.

Os principais problemas são:

- tecnologia ainda limitada (lento e caro às vezes)
- difícil de usar
- falta de regras claras
- muita gente ainda não entende ou não confia
