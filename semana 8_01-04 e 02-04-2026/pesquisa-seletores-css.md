# Resumo dos seletores css

| Seletor           | O que faz                      |
| ----------------- | ------------------------------ |
| `:first-child`    | Primeiro filho                 |
| `:last-child`     | Último filho                   |
| `:nth-last-child` | Conta de trás                  |
| `:has()`          | Seleciona pai que tem filho    |
| `:is()`           | Agrupa seletores               |
| `:where()`        | Agrupa sem aumentar prioridade |
